How pirate game was created...



This project all started in the summer of 2021 as stated before. It has become an ongoing project for the foreseeable future after. The code is primarily made up of JavaScript, python CSS and Html. Python acts as the server side for the website. This means all the databases  of concurrent games are stored on the sever and the roots for the website (URL/roots). The  HTML tells web browsers how to display a web page's words and images. This means all the images and words you see right now are written in Html - even this text! We have tried to make this website as adaptable as possible by using the size measurements as VH where possible meaning the layout should adapt  for the size of the view port  Html would look incredibly boring if it weren't for the   wonders of CSS. CSS is responsible for describing  the presentation of the web page. This  includes, colours, layout and fonts.  These are standard elements to a web page which you see day to day. Throughout our journey, we have taught ourselves how to you these languages on top of the most important part - Python Flask.  Python flask is sued to control the flow of data between client and server through web sockets.  In addition, sqlalchemy has been used to create the databases holding all the information for the games to function!
